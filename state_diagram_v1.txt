mentor_executor.py polls every 2 seconds:
  │
  ├─> HTTP GET: http://127.0.0.1:5000/api/signal_preview
  │
  └─> api_server.py receives request
       │
       ├─> EngineAdapter.generate_signal_package()
       │    │
       │    ├─> Reads config.commissioning.event_timeframes = ["M1"]
       │    ├─> Single-TF mode: calls engine.generate_signal_package(event_timeframe="M1")
       │    │
       │    └─> engine.py (v2.13.0)
       │         │
       │         ├─> Fetches M1 OHLCV data from MT5
       │         ├─> Calculates: BOS, ST, BB, ATR, regime, alignment
       │         ├─> Decision logic: BUY / SELL / HOLD
       │         ├─> Plan calculation: entry=close, sl/tp from ATR
       │         ├─> Blockers: vol_gate, st_relax, bos_break, rr_check
       │         │
       │         └─> Returns signal package:
       │              {
       │                "request_id": "REQ-GOLD-1234567890",
       │                "decision": "BUY" | "SELL" | "HOLD",
       │                "plan": {"entry": 2600.0, "sl": 2595.0, "tp": 2610.0},
       │                "metrics": {"rr": 2.0, "regime": "EXPANSION", ...},
       │                "blocked_by": [],
       │                "event_timeframe": "M1",
       │                ...
       │              }
       │
       └─> HTTP 200 JSON response back to mentor

  ├─> HTTP POST: http://127.0.0.1:5000/api/ai_confirm

       │    │

       │    │    - blocked_by present? → DENY
       │    │    - decision not in (BUY, SELL)? → DENY
       │    │    - plan invalid (sl/tp direction)? → DENY
       │    │    - rr < min_rr? → DENY
       │    │
       │    ├─> If use_llm=true (config.ai_confirm.use_llm):
       │    │    │
       │    │    ├─> Build LLM prompt (confirm-only, NO plan modification):
       │    │    │    "TASK: You are FINAL CONFIRM layer"
       │    │    │    "Direction is LOCKED; do not flip BUY/SELL"
       │    │    │    "Output JSON: {approved: bool, confidence: float, reason: str}"
       │    │    │
       │    │    ├─> HTTP POST to DeepSeek/OpenAI
       │    │    │    Model: deepseek-reasoner (default)
       │    │    │    Timeout: 8 sec
       │    │    │
       │    │    └─> Parse response:
       │    │         - Extract: approved, confidence, reason, bullets
       │    │         - NO confirmed_plan in response (Phase 6 verified)
       │    │
       │    └─> Return AI response:

       │           "approved": true,
       │           "confidence": 0.85,
       │           "reason": "Strong trend alignment, RR favorable",
       │           "bullets": ["Watch for reversal", "Trail stop at TP1"],
       │           "provider": "deepseek",
       │           "model": "deepseek-reasoner"
       │         }
       │
       └─> HTTP 200 JSON response

┌─────────────────────────────────────────────────────────────────────────────┐
│ ASYNC SIDECAR MONITOR (risk_guard_hardstop.py v1.1.1)                      │
└─────────────────────────────────────────────────────────────────────────────┘

Every 5 seconds (config.risk_guard.poll_interval_sec):
  │
  ├─> MT5 Account Snapshot:
  │    │
  │    ├─> mt5.initialize()
  │    ├─> acc = mt5.account_info()
  │    │    - equity: current account equity
  │    │    - balance: account balance
  │    │    - profit: floating P&L
  │    │    - margin_free: available margin
  │    │
  │    ├─> positions = mt5.positions_get()
  │    │    - open_positions: count
  │    │    - open_volume: sum of volumes
  │    │
  │    └─> mt5.shutdown()
  │
  ├─> Daily Drawdown Calculation:
  │    │
  │    ├─> Load state: .risk_guard_state.json
  │    │    - day_start_equity (stored at day start)
  │    │    - initial_equity (from config or first run)
  │    │
  │    ├─> daily_dd_pct = (day_start_equity - equity) / day_start_equity * 100
  │    └─> total_dd_pct = (initial_equity - equity) / initial_equity * 100
  │
  ├─> Breach Check:
  │    │
  │    ├─> If daily_dd_pct >= config.risk_guard.max_daily_dd_pct (2.0%):
  │    │    └─> breach.append("daily_dd_pct")
  │    │
  │    ├─> If total_dd_pct >= config.risk_guard.max_total_dd_pct (8.0%):
  │    │    └─> breach.append("total_dd_pct")
  │    │
  │    ├─> If open_positions > max_open_positions (if configured):
  │    │    └─> breach.append("max_open_positions")
  │    │
  │    └─> If open_volume > max_total_volume (if configured):
  │         └─> breach.append("max_total_volume")
  │
  ├─> If breach detected:
  │    │
  │    ├─> Write KILL_SWITCH.txt:
  │    │    Content: "RISK_GUARD_HARDSTOP
  │    │              breach=['daily_dd_pct']
  │    │              daily_dd_pct=2.34
  │    │              total_dd_pct=1.20
  │    │              utc=2026-03-13T10:30:00Z"
  │    │
  │    ├─> Log event: "kill_switch_written"
  │    │
  │    └─> Continue monitoring (does not exit)
  │
  └─> Log check result to logs/risk_guard.jsonl
